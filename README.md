# 계산기 만들기 프로젝트
> Kotlin 언어를 사용하여 간단한 계산기를 만들었다!
 
## 기능 설명
기본 사칙연산이 되는 계산기.

## 구현 해야하는 항목 (단계)
1. 더하기, 빼기, 나누기, 곱하기, 나머지 연산을 수행할 수 있는 Calculator 클래스를 만든다.<br>
2. 각각의 연산들을 class로 만든 후 Calculator 클래스와 관계를 맺는다.(상속)<br>
3. 구현한 연산 클래스들을 AbstractOperator라는 클래스명으로 만들어 사용하여 추상화하고 그에 맞게 전체 코드를 수정한다.
